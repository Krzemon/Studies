## Kompilacja i uruchamianie

Z użyciem standardowych skrótów CMake z VS Code. 

Ręcznie:

Z poziomu katalogu z `main.cpp` :

```
mkdir build
cd build
cmake ..
make
./lab01
cd ..
gnuplot script.gnu
```

Wykresy zostaną utworzone w katalogu `plot/`.
